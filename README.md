# Medusa

A WIP QuickJS wrapper with helper functions

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     medusa:
       github: grkek/medusa
   ```

2. Run `shards install`

3. Update the Boehm GC include path in the .envrc file and source from it

## Usage

1. Run `make test`
